# bright-coffee-Shop
Analysis for the coffee shop for the last six months

###Problem Statement:
The CEO wants to undersand what caused the decline in revenue in the past 6 months.
I am asked to to diagnostic analysis to understand the decline in revenue.

### AIm 
The aim of this project is to provide detailed insights on what caused the decline in revenue.

### Objectives
To achieve the aim the following objectives must be met:
- Use miro for maping and brainstorming.
- Use canva() to keep track of time to finish the project:
- Use Databricks for data processing.

### Summary of results
- The sales droped by 50% in January as compared in december this could be due seasonal changes
